다중 선택 + 사유 워드클라우드 설문 앱

이 페이지는 처음 접속한 사용자를 위한 시작 화면입니다. 아래 버튼으로 원하는 역할 페이지로 이동하세요.

페이지 바로가기

CLIENT (설문 설계/세션 관리) HOST (진행/종료 및 결과 공개) PARTICIPANT (참여자 응답) TEST (빠른 동작 확인)

처음 사용하는 경우 추천 순서

  1. CLIENT에서 Apps Script URL을 저장합니다.
  2. CLIENT에서 설문(선택지 2~5개)을 만들고 세션을 시작합니다.
  3. 참여자에게 PARTICIPANT 링크를 공유합니다.
  4. HOST에서 참여자 수를 보다가 종료 후 결과를 확인합니다.
